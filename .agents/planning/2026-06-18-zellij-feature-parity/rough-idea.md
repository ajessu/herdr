# Zellij Feature Parity for Herdr

## Context
User switched from Zellij to Herdr for workspace support and agent views. Missing many Zellij features around tab and pane management.

## Already Implemented
- Floating panes
- Quicker rendering

## Features of Interest
- Stacked panes
- Better shortcut visibility
- Alt-related shortcuts
- Shortcut visibility on right side of bottom bar
- Tab and pane management features generally

## Goal
Research Zellij features, assess herdr codebase readiness, and recommend which features to port based on difficulty and value.
