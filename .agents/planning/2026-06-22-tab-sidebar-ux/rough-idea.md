# Tab Bar, Bottom Bar & Sidebar UX Improvements

Improve herdr's tab bar, bottom bar, and sidebar UX to match zellij's superior design:

## Tab Bar
- Tab overflow: show grouped/batch navigation (not one-by-one scroll) — copy zellij's approach where it shows a "[+N]" bunch you can jump to
- Tab display: better visual styling/painting
- Active tab indicators matching zellij's approach

## Bottom Bar / Hint Bar
- Left side: Ctrl shortcuts (already partially done)
- Right side: Alt shortcuts (currently missing)
- Visual display/painting matching zellij's cleaner style

## Sidebar
- Better resize handles — more prominent, duplicated (not only at bottom)
- Mobile/touch friendly sizing
- Better minimize/expand UX

Key constraint: literally copy zellij's tab overflow behavior (batch navigation, not one-by-one).
